# vim/neovim/tmux dotfiles

### Install python3 and pip3
`brew install vim`

### Install vim8 with python3 and timers support
`brew install python3`

### Install nvim 
`brew install neovim`
`pip3 install neovim`

### Install ctags 
`brew install ctags`

### Create sym links
